# 全方位智慧貼身照護平台藍圖

---

## 1. 客戶端 (LINE@)

> 這是面向終端用戶與家屬的介面，主要功能包含：

### 1.1 個案帳號綁定
- 輸入個案邀請碼
- 發送驗證碼至手機
- 驗證碼確認綁定
- 綁定成功/失敗訊息提示
- 重新綁定流程

### 1.2 個案家人綁定
- 輸入家人關係 (父母/子女/配偶/其他)
- 輸入被綁定者的個案編號
- 發送綁定邀請通知給個案
- 個案同意/拒絕綁定
- 家人綁定列表管理
- 解除家人綁定

### 1.3 首頁
- **服務類別**
  - 服務類別列表
    - 類別圖示顯示
    - 類別名稱顯示
    - 類別排序 (依平台端設定)
  - 服務項目列表
    - 服務圖片
    - 服務名稱
    - 服務簡述
    - 服務價格
    - 上架狀態篩選 (僅顯示已上架)
  - 服務詳情頁
    - 服務圖片輪播
    - 服務名稱
    - 服務價格 (原價/促銷價)
    - 服務詳細說明 (富文本)
    - 規格選擇 (若有多規格)
    - 數量選擇
    - 加入購物車
    - 立即購買
    - 收藏服務
  - 服務搜尋
    - 關鍵字搜尋
    - 類別篩選
    - 價格區間篩選
    - 排序 (價格/上架時間/熱門)
  - 購物車
    - 購物車列表
    - 數量調整
    - 刪除項目
    - 小計計算
    - 前往結帳
  - 結帳流程
    - 確認訂購項目
    - 填寫/選擇收件資訊
    - 選擇付款方式
    - 優惠券/折扣碼輸入
    - 訂單金額確認
    - 確認送出訂單
    - 付款處理
    - 訂購完成頁面
- **機構功能**
  - 個案關懷通訊
    - 即時文字訊息
    - 圖片/檔案傳送
    - 語音訊息
    - 已讀/未讀狀態
    - 歷史訊息查詢
    - 訊息通知推播
  - ?? (待定)

### 1.4 訊息小鈴鐺
- 通知訊息列表
  - 系統公告通知
  - 訂單狀態變更通知
  - 服務預約提醒
  - 機構訊息通知
  - 付款相關通知
- 未讀訊息計數顯示
- 標記已讀/全部已讀
- 刪除通知
- 通知詳情檢視
- 點擊通知跳轉對應頁面

### 1.5 訂購清單
- 訂單列表
  - 依狀態篩選 (待付款/處理中/已完成/已取消)
  - 依日期區間篩選
  - 依服務類型篩選
- 訂單詳情
  - 訂單編號
  - 訂購時間
  - 商品/服務明細
  - 金額明細 (小計/運費/折扣/總計)
  - 收件資訊
  - 付款方式/狀態
  - 物流狀態追蹤
- 訂單操作
  - 取消訂單
  - 再次購買
  - 申請退款/退貨
  - 聯繫客服
  - 評價服務

### 1.6 個人資訊
- **切換帳號**
  - 已綁定帳號列表
  - 快速切換個案身份
  - 顯示當前操作身份
  - 各帳號獨立資料顯示
- **資料修改**
  - 姓名編輯
  - 聯絡電話修改
  - 電子郵件修改
  - 通訊地址編輯
  - 緊急聯絡人設定
  - 頭像上傳
  - 生日設定
  - 偏好語言設定
- **個案關懷者綁定**
  - 新增關懷者
  - 輸入關懷者資訊 (姓名/電話/關係)
  - 發送綁定邀請
  - 關懷者列表管理
  - 設定主要關懷者
  - 移除關懷者
- **付款綁定**
  - 新增付款方式
  - 信用卡綁定 (卡號/有效期/CVV)
  - LINE Pay 綁定
  - 銀行帳戶綁定
  - 設定預設付款方式
  - 付款方式列表
  - 移除付款方式

### 1.7 聯絡我們
- 常見問題 FAQ
- 客服表單提交
  - 選擇問題類型
  - 問題描述輸入
  - 上傳附件/截圖
  - 聯絡方式填寫
- 客服專線撥打
- 客服 LINE 連結
- 營業時間說明
- 歷史客服單查詢
- 客服單回覆通知

## 2. 機構端

> 這是提供給合作機構（如照護中心、服務供應商）使用的後台管理介面：

### 2.1 個案會員管理
- 個案綁定設定
  - 綁定邀請碼管理
    - 產生個案專屬邀請碼
    - 邀請碼列表查詢
    - 作廢邀請碼
  - 綁定審核
    - 待審核綁定申請列表
    - 審核通過/駁回
- 個案會員列表
  - 分頁顯示
  - 關鍵字搜尋 (姓名/電話/編號)
  - 進階篩選 (狀態/加入日期/服務類型)
  - 排序功能
  - 匯出 Excel
- 個案資訊查詢
  - 基本資料檢視
  - 健康狀況紀錄
  - 服務使用歷史
  - 關懷者資訊
  - 付款紀錄
  - 通訊紀錄
- 個案狀態管理
  - 啟用/停用個案
  - 狀態變更紀錄
  - 備註新增

### 2.2 訂購清單查詢 (分潤)
- 訂單列表查詢
  - 依日期區間查詢
  - 依訂單狀態篩選
  - 依服務類型篩選
  - 依個案篩選
  - 訂單詳情檢視
- 分潤報表
  - 月份分潤明細
  - 分潤比例顯示
  - 待結算金額
  - 已結算金額
  - 分潤計算明細
- 收益統計
  - 日/週/月/年報表
  - 圖表視覺化 (折線圖/長條圖)
  - 同期比較分析
  - 服務類型收益占比
  - 報表匯出 (PDF/Excel)

### 2.3 機構功能
- **機構資訊管理**
  - 機構基本資料維護
    - 機構名稱
    - 統一編號
    - 負責人姓名
    - 聯絡電話
    - 電子郵件
    - 機構地址
    - 機構簡介
    - 機構 Logo 上傳
  - 營業資訊設定
    - 營業時間設定
    - 服務區域設定
    - 服務項目設定
    - 定價設定
    - 休假日設定
- **管理者權限管理**
  - 管理者帳號管理
    - 新增管理者
    - 編輯管理者資訊
    - 重設密碼
    - 停用/啟用帳號
    - 刪除管理者
  - 角色權限設定
    - 建立角色 (管理員/一般人員/客服等)
    - 角色權限配置
    - 指派角色給管理者
    - 權限項目勾選
- **機構公告管理**
  - 公告列表
  - 新增公告
    - 公告標題
    - 公告內容 (富文本編輯)
    - 公告類型 (一般/重要/緊急)
    - 發布對象 (全部個案/指定個案)
    - 發布時間設定
    - 附件上傳
  - 編輯公告
  - 刪除公告
  - 公告上架/下架
  - 公告閱讀統計
- **個案關懷通訊功能**
  - 通訊對象列表
  - 即時通訊介面
    - 文字訊息傳送
    - 圖片/檔案傳送
    - 罐頭訊息快捷發送
    - 訊息已讀回條
  - 群發訊息
  - 通訊紀錄查詢
    - 依個案查詢
    - 依日期查詢
    - 關鍵字搜尋
  - 通訊紀錄匯出

### 3.1 機構管理
- 機構列表
  - 分頁顯示
  - 關鍵字搜尋

  - 待審核列表

  - 駁回原因填寫
